# ChatRoomVue - 聊天室前端

基于 Vue 3 + Vite 的聊天室前端 SPA，支持动态主题、音乐同步、文件上传、头像更换等功能。

## 技术栈
- Vue 3
- Vue Router
- Vite
- SignalR Client
- js-sha256

## 功能特性
- 用户登录/注册/找回密码
- 实时聊天界面（文字、图片、文件、表情）
- 全局背景音乐控制（播放、暂停、切歌、循环）
- 播放列表管理
- 在线用户列表及 @提及
- 头像上传与实时同步
- 系统公告栏
- 主题系统（内置三款，支持导入 CSS 文件）
- 文件上传进度条

## 环境要求
- Node.js 18+ / 20 LTS
- npm 或 yarn
